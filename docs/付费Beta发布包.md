## 1. 发布判断

我的判断：可以现在推出去试着赚钱，但必须用“付费 Beta / 早鸟支持者”的口径，不应包装成正式商业版。

原因：

- TypeWhale 的核心体验已经能给真实用户创造价值：在任何应用里快捷键录音、本地识别、胶囊反馈、自动粘贴，并可选智能整理/翻译。
- 当前商业 P0 仍未完全闭环：Developer ID 公证、自动更新、正式授权体系、隐私政策/EULA、模型商业再分发确认。
- 因此现在卖的不是“完全成熟的正式版”，而是“早鸟席位 + 当前可用 Beta + 后续正式版权益”。

推荐口径：

```text
TypeWhale 正在开放一小批 Mac 付费 Beta 早鸟席位。
你现在购买的是早期使用权和正式版权益，不是已经完全商业化的稳定版。
如果 7 天内无法在你的日常工作流里产生价值，可以退款。
```

## 3. 定价

### 推荐首发价

```text
早鸟价：99 元 / 人
名额：前 30 人
退款：7 天内不满意可退
```

为什么不是免费：

- 免费用户会给“有人下载”的错觉，但不能验证付费意愿。

早鸟用户权益：

- 粘贴错目标或破坏剪贴板被反馈。
- 用户普遍认为输出不可用或曲解意思。

暂停不是放弃商业化，而是停止放大坏体验。

## 14. 对外 FAQ

### 它会上传我的语音吗？

语音识别默认在本机完成。智能整理、翻译和截图翻译属于可选增强能力，只有你启用并配置 API Key 后才会调用第三方模型。

### 为什么 macOS 会提示无法验证开发者？

当前是 Beta，还没有完成 Developer ID 公证。你需要在系统设置的隐私与安全性里手动允许。正式商业版会补齐公证，让安装体验更顺。

### 付款后我买到什么？

你买到的是 TypeWhale macOS 个人版早鸟席位：当前 Beta 使用权、正式个人版权益、1 年更新和优先反馈通道。

### 不好用能退吗？

可以。7 天内如果没有进入你的真实工作流，或者安装后无法正常使用，可以退款。

### 它和会议转写工具有什么区别？

TypeWhale 不是会议转写工具。它更像一个桌面语音输入工具：你在任何应用的输入框里说话，结果直接回到当前应用。
